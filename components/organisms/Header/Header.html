🎯 ViberAiden Header Component

나인원랩스 디자인 시스템을 기반으로 ViberAiden 브랜딩에 최적화된 Header 컴포넌트입니다. 80px 고정 높이, 반응형 네비게이션, 접근성을 완벽 지원합니다.

  • 80px 고정 높이 헤더 with 스크롤 효과
  • ViberAiden 브랜딩 메뉴 (문제들, 해결책, 기획자 이야기, 연락하기)
  • 반응형 햄버거 메뉴 (1024px 이하)
  • 나인원랩스 Button 컴포넌트 통합
  • 완전한 키보드 네비게이션 및 스크린 리더 지원
  • 부드러운 애니메이션 및 호버 효과

🔧 사용 방법

기존 Button 컴포넌트와 Typography 시스템을 활용하여 구성된 Header입니다.

<header class="header" id="header"> <div class="header-container"> <a href="#" class="header-logo">ViberAiden</a> <nav class="header-nav"> <ul class="nav-links"> <li><a class="nav-link active">문제들</a></li> <li><a class="nav-link">해결책</a></li> <li><a class="nav-link">기획자 이야기</a></li> <li><a class="nav-link">연락하기</a></li> </ul> <div class="header-cta"> <a class="btn btn-primary">해결한 문제들 보기</a> </div> </nav> </div> </header>

🎨 인터랙션 테스트

다음 기능들을 테스트해보세요:

  • 스크롤 효과: 페이지를 스크롤하면 헤더 배경이 반투명해집니다
  • 키보드 네비게이션: Tab 키로 모든 요소에 접근 가능
  • 반응형 메뉴: 브라우저 크기를 조정하여 모바일 메뉴 확인
  • 호버 효과: 로고, 메뉴 링크, 버튼에 마우스를 올려보세요

📝 스크롤 테스트용 콘텐츠

이 섹션은 스크롤 효과를 테스트하기 위한 더미 콘텐츠입니다.
