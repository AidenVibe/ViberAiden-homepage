🚀 ViberAiden Hero Component

나인원랩스 디자인 시스템을 기반으로 ViberAiden 브랜딩에 최적화된 Hero 컴포넌트입니다. 풀스크린 높이, 그라데이션 배경, 타이핑 애니메이션을 지원합니다.

  • 풀스크린 Hero 섹션 with 나인원랩스 그라데이션
  • ViberAiden 핵심 메시지: "사람들이 느끼지만 말하지 않는 불편함을 찾는 기획자"
  • 타이핑 효과 애니메이션 ("말하지 않는 불편함")
  • 순차적 페이드인 애니메이션 (제목 → 부제목 → 설명 → CTA)
  • 통계 카드 with 카운트업 애니메이션
  • 반응형 디자인 (Desktop → Tablet → Mobile → Small Mobile)
  • 접근성: reduced-motion 지원, 적절한 ARIA 레이블

⚡ 애니메이션 효과

페이지 로드 시 다양한 애니메이션 효과가 순차적으로 실행됩니다.

  • 배경 패턴: 부드러운 플로팅 애니메이션 (20초 사이클)
  • 타이핑 효과: "말하지 않는 불편함" 텍스트의 깜빡이는 커서
  • 순차 페이드인: 0.2s → 0.4s → 0.6s → 0.8s 간격으로 요소 표시
  • 카운트업: 통계 숫자가 0에서 목표값까지 애니메이션
  • 스크롤 인디케이터: 바운스 애니메이션으로 스크롤 유도

📱 반응형 브레이크포인트

다양한 디바이스에서 최적화된 경험을 제공합니다.

  • Large (1280px+): hero-title 8xl, hero-subtitle 3xl
  • Desktop (1024px+): 기본 크기, 통계 4열 그리드
  • Tablet (≤1023px): hero-title 6xl, 세로 CTA 배치
  • Mobile (≤768px): hero-title 5xl, 통계 2열 그리드
