줄 간격 변형

typo-leading-tight (1.25)

Tight line height - 제목이나 짧은 텍스트에 적합합니다. 줄 간격이 좁아 밀도감 있는 레이아웃을 만듭니다.

typo-leading-snug (1.375)

Snug line height - 일반적인 텍스트에 사용됩니다. 가독성과 밀도 사이의 균형을 맞춥니다.

typo-leading-normal (1.5)

Normal line height - 기본 줄 간격입니다. 대부분의 본문 텍스트에 사용하기 적합합니다.

typo-leading-relaxed (1.625)

Relaxed line height - 여유로운 줄 간격으로 긴 텍스트의 가독성을 향상시킵니다.

typo-leading-loose (2.0)

Loose line height - 가장 넓은 줄 간격입니다. 특별한 디자인 효과나 접근성 개선에 사용됩니다.

접근성 가이드라인

✓ 색상 대비 (4.5:1 이상)

모든 텍스트는 배경과 4.5:1 이상의 대비율을 유지합니다.
Primary 색상: 7.2:1
Secondary 색상: 8.1:1
Muted 색상: 4.6:1

✓ 헤딩 계층구조

의미적 HTML 구조를 준수하여 스크린 리더 호환성을 보장합니다.
h1 → h2 → h3 → h4 → h5 → h6 순서로 사용

✓ 최소 폰트 크기

본문 텍스트는 최소 14px (typo-base) 이상 사용을 권장합니다.
모바일에서는 16px (typo-md) 이상 사용

✓ 충분한 줄 간격

본문 텍스트의 줄 간격은 1.5 (typo-leading-normal) 이상을 사용합니다.
긴 텍스트의 경우 1.625 (typo-leading-relaxed) 권장
