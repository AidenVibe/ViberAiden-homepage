접근성 체크리스트

📋 필수 요구사항

  • Label 연결: 모든 Input은 label 요소와 for 속성으로 연결
  • Required 표시: 필수 입력 필드는 required 속성과 시각적 표시(*)
  • Error 메시지: aria-invalid와 aria-describedby로 에러 메시지 연결
  • 키보드 네비게이션: Tab, Enter, Space 키로 모든 기능 접근 가능
  • 포커스 인디케이터: 포커스 시 명확한 시각적 표시

🎯 권장 사항

  • 터치 타겟: 최소 44×44px 크기 보장
  • 색상 대비: 텍스트와 배경 간 4.5:1 대비율 이상
  • 도움말 텍스트: 복잡한 입력 필드에는 명확한 설명 제공
  • Fieldset 사용: 관련 Input 그룹은 fieldset과 legend로 묶기
  • Placeholder 주의: 중요한 정보는 레이블이나 도움말에 포함

❌ 피해야 할 것들

  • Placeholder만으로 레이블 대체하기
  • 색상만으로 상태 표현하기 (텍스트 설명 병행 필요)
  • 너무 작은 터치 타겟 (44px 미만)
  • 포커스 아웃라인 제거하기
  • 자동 완성 차단 (보안이 필요한 경우 제외)
